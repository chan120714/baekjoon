### 문제 설명

<p>$1$ 이상 $N$ 이하의 정수가 순서대로 적힌 $N$장의 카드가 있다. 각각의 카드는 초기에 자기 자신만으로 이루어진 덱이다.</p>

<p>와파스는 덱을 합칠 때마다 종이에다 수를 적는다. 처음에는 종이에 $1$이 적혀있다.</p>

<p>와파스는 다음 규칙을 따라 두 덱을 골라 하나의 덱으로 합친다.</p>

<ul>
	<li>두 덱을 합치기 전, 두 덱에서 카드를 각각 한 장씩 뽑는다.</li>
	<li>뽑은 두 카드에 적힌 수를 $a$, $b$라고 했을 때, $a+b$가 반드시 제곱수여야 한다. 이러한 카드를 뽑는 것이 불가능하다면 이 두 덱은 합칠 수 없다.</li>
	<li>두 덱을 하나로 합친 후에는 종이에 $|a - b|$를 적는다.</li>
</ul>

<p>$N$장의 카드를 하나의 덱으로 합친 후, 와파스는 종이에 적힌 $N$개의 수를 모두 곱한 값인 $x$를 구한다.</p>

<p>와파스는 $x$의 값을 최소로 하여 덱을 모두 합치고 싶어한다. 와파스를 위해 $x$의 최솟값을 구해보자.</p>

### 입력 

 <p>양의 정수 $N$이 주어진다. $(1 \le N \le 10^{14})$</p>

### 출력 

 <p>$x$의 최솟값을 $998\,244\,353$로 나눈 나머지를 출력한다.</p>

<p>만약 $N$장의 카드를 하나의 덱으로 합칠 수 없다면 <span style="color:#e74c3c;"><code>-1</code></span>을 출력한다.</p>
